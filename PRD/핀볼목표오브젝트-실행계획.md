# 핀볼 목표 오브젝트 실행계획

## 요구사항 요약

**요구사항**: 핀볼 게임에서 목표 오브젝트(target) 생성

**목적**: 사용자에게 명확한 목표를 부여하여 도전 정신을 고취시키고 게임을 계속하게 만듦

## 현재상태 분석

**기존 bumper 구조** (Pinball.jsx:123-128):
- `Bodies.circle(400, 600, 40)`: 원형 오브젝트, 위치 (400, 600), 반지름 40
- `isStatic: true`: 고정된 위치
- `restitution: 1.5`: 충돌 시 강하게 튕겨냄
- `label: 'bumper'`: 충돌 감지를 위한 식별자
- `render: { fillStyle: '#e74c3c' }`: 빨간색

**기존 bumper 충돌 이벤트** (Pinball.jsx:206-218):
- `bodyA.label === 'bumper'` 또는 `bodyB.label === 'bumper'`로 bumper 충돌 감지
- 충돌 시 공에 추가 속도 부여 (Vector 계산 사용)

## 구현 방법

**기술적 접근**:
- bumper 코드를 복사하여 target 생성
- label을 'target'으로 변경
- 색상을 하늘색(#87CEEB)으로 변경
- restitution은 bumper와 동일하게 1.5 유지 (충돌 시 튕겨냄)
- 충돌 이벤트에 target 충돌 감지 로직 추가 (콘솔 메시지만 출력)

**배치 위치**:
- bumper와 다른 위치에 배치 (예: (500, 500))

## 구현 단계

### 1. target Bodies 생성

```javascript
// 목표 오브젝트 만들기 (충돌 시 강하게 튕겨냄)
const target = Bodies.circle(500, 500, 40, {
  isStatic: true,
  restitution: 1.5,
  label: 'target',
  render: { fillStyle: '#87CEEB' }
});
```

- 위치: x=500, y=500 (bumper와 다른 위치)
- 반지름: 40 (bumper와 동일)
- `isStatic: true`: 고정된 위치
- `restitution: 1.5`: bumper와 동일한 반발력
- `label: 'target'`: 충돌 감지를 위한 식별자
- `fillStyle: '#87CEEB'`: 하늘색 (Sky Blue)

### 2. World에 target 추가

```javascript
// 세계에 모든 물체 추가
World.add(engine.world, [
  ground,
  leftWall,
  rightWall,
  upWall,
  deathZone,
  ball,
  obstacle1,
  obstacle2,
  bumper,
  target,  // target 추가
  leftFlipper,
  rightFlipper,
  leftFlipperConstraint,
  rightFlipperConstraint
]);
```

- `World.add()` 배열에 `target` 추가
- bumper 다음에 배치하여 가독성 향상

### 3. target 충돌 이벤트 추가

```javascript
// 공과 목표 오브젝트가 충돌했는지 확인
if ((bodyA.label === 'target' && bodyB === ball) ||
    (bodyB.label === 'target' && bodyA === ball)) {
  console.log('target 충돌했음');
}
```

- bumper 충돌 이벤트와 동일한 구조
- `bodyA.label === 'target'` 또는 `bodyB.label === 'target'`로 충돌 감지
- 콘솔에 "target 충돌했음" 메시지 출력
- bumper 충돌 이벤트 아래에 추가 (동일한 `Events.on('collisionStart')` 내부)
