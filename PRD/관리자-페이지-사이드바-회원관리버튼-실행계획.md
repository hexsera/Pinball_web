# 관리자 페이지 사이드바 회원관리 버튼 실행계획

## 요구사항 요약

**요구사항**: 관리자 페이지 사이드바에 "회원관리" 버튼을 추가하고, 클릭 시 회원관리 페이지로 이동

**목적**: 현재 회원관리 페이지(/admin/users)로 이동할 수 있는 수단이 없어 접근이 불가능한 상태이므로, 사이드바에 네비게이션 버튼을 추가하여 편의성 향상

## 현재상태 분석

- AdminSidebar.jsx는 현재 임시 이미지(admin_sid_temp.png)만 표시하고 있음
- 실제 동작하는 버튼이 없는 상태 (순수 이미지 기반)
- App.jsx에 `/admin/users` 라우트와 AdminUserPage 컴포넌트는 이미 구현되어 있음
- MUI(Material-UI) 환경 사용 중
- 사이드바 너비: 270px, 영구 표시(permanent) Drawer 사용

## 구현 방법

React Router의 `useNavigate` 훅과 MUI의 `List`, `ListItem`, `ListItemButton`, `ListItemIcon`, `ListItemText` 컴포넌트를 사용하여 네비게이션 버튼 구현. 임시 이미지를 제거하고 Dashboard와 동일한 스타일의 버튼 목록으로 교체. Dashboard의 사이드바 스타일(hover 효과, 선택 상태 하이라이트, 아이콘 포함)을 동일하게 적용.

## 구현 단계

### 1. AdminSidebar.jsx에 필요한 import 추가

```javascript
import { Box, Drawer, List, ListItem, ListItemButton, ListItemIcon, ListItemText } from '@mui/material';
import { People } from '@mui/icons-material';
import { useNavigate } from 'react-router-dom';
```

- React Router의 `useNavigate` 훅을 사용하여 페이지 이동 구현
- MUI의 List 관련 컴포넌트(ListItemIcon 포함)를 사용하여 사이드바 버튼 목록 생성
- MUI Icons에서 `People` 아이콘 import (회원관리 아이콘)
- CardMedia import는 제거 (임시 이미지 사용 중단)

### 2. useNavigate 훅과 선택 상태 관리

```javascript
function AdminSidebar() {
  const navigate = useNavigate();
  const [selectedIndex, setSelectedIndex] = useState(0);

  return (
    // ...
  );
}
```

- `useNavigate` 훅을 컴포넌트 내부에서 선언하여 페이지 이동 함수 확보
- `selectedIndex` 상태로 현재 선택된 메뉴 추적 (Dashboard와 동일한 패턴)
- navigate 함수를 버튼 클릭 핸들러에서 사용

### 3. 임시 이미지 제거 및 Dashboard 스타일 버튼 구현

```javascript
<Drawer
  variant="permanent"
  sx={{
    display: { xs: 'block', sm: 'block' },
    '& .MuiDrawer-paper': {
      boxSizing: 'border-box',
      width: drawerWidth + 10,
      backgroundColor: '#ffffffff',
      border: '1px solid #e5e7eb',
    },
  }}
  open
>
  <Box sx={{ width: '100%' }}>
    <List sx={{ px: 2 }}>
      <ListItem disablePadding>
        <ListItemButton
          onClick={() => {
            setSelectedIndex(0);
            navigate('/admin/users');
          }}
          sx={{
            borderRadius: 1,
            mb: 0.5,
            color: '#2f3135ff',
            '&:hover': {
              backgroundColor: '#eff3f8ff',
            },
            ...(selectedIndex === 0 && {
              backgroundColor: '#ECF3FF',
              color: '#465FFF',
              '&:hover': {
                backgroundColor: '#ECF3FF',
              },
            }),
          }}
        >
          <ListItemIcon sx={{ color: 'inherit', minWidth: 40 }}>
            <People />
          </ListItemIcon>
          <ListItemText primary="회원관리" />
        </ListItemButton>
      </ListItem>
    </List>
  </Box>
</Drawer>
```

- CardMedia로 표시하던 임시 이미지를 제거
- Dashboard와 동일한 스타일 적용:
  - borderRadius: 1 (둥근 모서리)
  - hover 시 배경색 #eff3f8ff
  - 선택 시 배경색 #ECF3FF, 텍스트색 #465FFF (파란색 하이라이트)
  - 기본 텍스트색 #2f3135ff
- ListItemIcon으로 People 아이콘 추가 (minWidth: 40)
- onClick에서 selectedIndex 업데이트 및 페이지 이동 처리
- px: 2로 좌우 여백 확보 (Dashboard와 동일)

### 4. 추가 버튼 확장 가능한 구조 (선택사항)

```javascript
import { useState } from 'react';
import { Dashboard as DashboardIcon, People } from '@mui/icons-material';

const menuItems = [
  { text: '대시보드', icon: <DashboardIcon />, path: '/admin' },
  { text: '회원관리', icon: <People />, path: '/admin/users' },
];

// 컴포넌트 내부
<List sx={{ px: 2 }}>
  {menuItems.map((item, index) => (
    <ListItem key={index} disablePadding>
      <ListItemButton
        onClick={() => {
          setSelectedIndex(index);
          navigate(item.path);
        }}
        sx={{
          borderRadius: 1,
          mb: 0.5,
          color: '#2f3135ff',
          '&:hover': {
            backgroundColor: '#eff3f8ff',
          },
          ...(selectedIndex === index && {
            backgroundColor: '#ECF3FF',
            color: '#465FFF',
            '&:hover': {
              backgroundColor: '#ECF3FF',
            },
          }),
        }}
      >
        <ListItemIcon sx={{ color: 'inherit', minWidth: 40 }}>
          {item.icon}
        </ListItemIcon>
        <ListItemText primary={item.text} />
      </ListItemButton>
    </ListItem>
  ))}
</List>
```

- 향후 확장을 위해 menuItems 배열로 구조화 (Dashboard와 동일한 패턴)
- 대시보드, 회원관리 등 여러 메뉴 항목 추가 가능
- map 함수로 반복 렌더링하여 코드 중복 제거
- selectedIndex로 현재 선택된 메뉴 하이라이트 자동 처리
