# 핀볼 화면 불필요한 세로 스크롤 제거 실행계획

## 요구사항 요약

**요구사항**: 핀볼 게임 화면에서 발생하는 불필요한 세로 스크롤을 제거한다.

**목적**: CSS `transform: scale()`은 요소를 시각적으로 줄여 보이게 하지만, DOM 레이아웃 공간은 원본 크기(1296px)를 그대로 유지한다. 이로 인해 화면보다 큰 레이아웃 공간이 생겨 스크롤바가 나타난다.

## 현재상태 분석

- `PinballPage.jsx`에서 `gameScale`을 계산해 `transform: scale(gameScale)` 적용
- `Pinball.jsx` 내부 캔버스는 `BASE_WIDTH=816`, `BASE_HEIGHT=1296` 고정 크기
- `scale`이 0.7이 되어도 레이아웃 공간은 여전히 816×1296px을 차지함
- 외부 컨테이너에 `minHeight: '100vh'`가 설정되어 있어 세로 스크롤 발생

## 구현 방법

`transform: scale()`의 레이아웃 공간 문제는 `width`/`height`를 `scale` 비율로 직접 줄이고, `transformOrigin: 'top center'`와 함께 음수 margin을 적용하여 해결한다.
`scale(S)`로 줄인 요소의 원본 크기가 W×H이면, 실제 차지해야 할 공간은 `W*S × H*S`이므로 `margin-bottom = H*(S-1)`, `margin-right = W*(S-1)` (음수값)을 적용한다.

## 구현 단계

### 1. 게임 컨테이너 외부 Box에 height 명시

```jsx
<Box sx={{
  flexGrow: 1,
  display: 'flex',
  justifyContent: 'center',
  alignItems: 'flex-start',
  overflow: 'hidden',   // 스케일 축소 후 남는 공간 잘라냄
  height: `calc(100vh - 64px)`,  // AppBar 높이(64px) 제외한 뷰포트 높이
}}>
```
- `overflow: 'hidden'`으로 스케일 변환 후 남는 레이아웃 공간이 스크롤을 유발하지 않도록 차단
- `height`를 명시해 flexGrow가 뷰포트를 벗어나지 않도록 제한

### 2. 스케일 적용 Box에 margin 보정 추가

```jsx
const scaledWidth = 816 * gameScale;
const scaledHeight = 1296 * gameScale;

<Box sx={{
  transform: `scale(${gameScale})`,
  transformOrigin: 'top center',
  width: 816,
  height: 1296,
  marginBottom: `${scaledHeight - 1296}px`,  // 음수값: 줄어든 만큼 여백 제거
  marginLeft: 'auto',
  marginRight: 'auto',
}}>
```
- `width`, `height`는 원본 크기 고정 (scale이 DOM 크기를 바꾸지 않으므로)
- `marginBottom = scaledHeight - 1296`은 scale < 1이면 음수가 되어 아래 여백을 제거
- `transformOrigin: 'top center'` 기준이므로 좌우는 `marginLeft/Right: auto`로 중앙 정렬 유지

### 3. 최상위 Box의 minHeight 제거

```jsx
<Box sx={{
  display: 'flex',
  flexDirection: 'column',
  height: '100vh',         // minHeight → height로 변경
  overflow: 'hidden',
  backgroundColor: '#0F172A'
}}>
```
- `minHeight: '100vh'`는 컨텐츠가 길면 페이지를 늘려 스크롤을 허용함
- `height: '100vh'`와 `overflow: 'hidden'`으로 뷰포트 크기를 고정하여 스크롤 발생 원천 차단

## 완료 체크리스트

- [ ] 화면을 세로로 스크롤해도 스크롤바가 나타나지 않는다
- [ ] 게임 캔버스가 화면 세로 크기에 맞게 자동으로 축소된다
- [ ] 게임 캔버스가 화면 가로 중앙에 위치한다
- [ ] 브라우저 창 크기를 조절해도 스크롤이 생기지 않는다
- [ ] 모바일(작은 화면)에서도 스크롤 없이 전체 게임이 보인다
