# 대쉬보드 페이지 단축 실행계획

## 요구사항 요약

**요구사항**:
1. "메인페이지" 메뉴 제거 → Dashboard 접속 시 "게임하기" 영역이 기본으로 표시
2. 사이드바에서 "소식", "통계" 메뉴 제거
3. 헤더의 검색바 제거

**목적**: 불필요한 UI 요소를 제거하여 핵심 기능(게임하기, 친구, 계정)만 남기고 UX를 단순화한다.

## 현재상태 분석

- `menuItems` 배열: 메인페이지, 게임하기, 친구, 소식, 통계, 계정 총 6개 항목
- `showPinball` 초기값 `false` → 기본 화면이 `mainele`(메인페이지 콘텐츠)로 표시됨
- 헤더 Toolbar에 `TextField`(검색바)가 포함되어 있음
- `handleMenuClick`에 소식/통계 처리 로직 존재

## 구현 방법

- `menuItems` 배열에서 불필요한 항목 제거 (소식, 통계, 메인페이지)
- `showPinball` 초기값을 `true`로 변경하여 게임하기 화면이 기본 표시되도록 변경
- `menuIndex` 초기값을 게임하기 항목 인덱스(0)에 맞게 조정
- 헤더 Toolbar에서 `TextField` 검색바 JSX 제거

## 구현 단계

### 1. showPinball 초기값 변경 및 menuIndex 초기값 조정

```javascript
const [showPinball, setShowPinball] = useState(true);
const [menuIndex, setMenuIndex] = useState(0);
```
- **무엇을 하는가**: Dashboard 접속 시 기본 화면을 "게임하기"로 변경
- `showPinball`을 `true`로 설정하면 조건부 렌더링 `showPinball ? <Pinball />` 가 즉시 실행됨
- `menuIndex`는 0으로 유지 (메인페이지 제거 후 게임하기가 첫 번째 항목이 됨)

### 2. menuItems 배열에서 불필요한 항목 제거

```javascript
const menuItems = [
  { text: '게임하기', icon: <CalendarMonth />, path: '/calendar' },
  { text: '친구', icon: <Person />, path: '/profile' },
  { text: '계정', icon: <Lock />, submenu: ['Sign In', 'Sign Up', 'Reset Password'] },
];
```
- **무엇을 하는가**: 사이드바에 표시되는 메뉴를 게임하기, 친구, 계정 3개로 줄임
- 메인페이지, 소식, 통계 항목을 배열에서 삭제
- 배열 순서가 변경되므로 menuIndex 초기값(0)이 게임하기를 가리키게 됨

### 3. handleMenuClick에서 불필요한 분기 제거

```javascript
function handleMenuClick(itemtext, index) {
  setMenuIndex(index);
  if (itemtext == '게임하기') {
    setShowPinball(true);
    setShowUserInfo(false);
    setShowFriend(false);
  }
  if (itemtext == '친구') {
    if (isLoggedIn == false) { navigate('/login') }
    else {
      setShowPinball(false);
      setShowUserInfo(false);
      setShowFriend(true);
    }
  }
  if (itemtext == '계정') {
    if (isLoggedIn == false) { navigate('/login') }
    else {
      setShowPinball(false);
      setShowUserInfo(true);
      setShowFriend(false);
    }
  }
}
```
- **무엇을 하는가**: 제거된 메뉴(메인페이지, 소식, 통계) 관련 if 분기를 삭제하여 코드 정리
- 메인페이지, 소식, 통계 분기 제거

### 4. 헤더 검색바(TextField) 제거

```jsx
<Box sx={{ display: 'flex', alignItems: 'center', gap: 2 }}>
  <IconButton
    color="inherit"
    edge="start"
    onClick={handleDrawerToggle}
    sx={{ mr: 2, display: { sm: 'none' } }}
  >
    <MenuIcon />
  </IconButton>
</Box>
```
- **무엇을 하는가**: 헤더에서 검색 기능 UI를 완전히 제거
- `TextField` 컴포넌트와 감싸는 Box의 `gap: 2` 속성 정리
- `Search` 아이콘 import도 사용하지 않으므로 import 목록에서 제거

### 5. 사용하지 않는 import 정리

```javascript
import {
  Menu as MenuIcon,
  Dashboard,
  CalendarMonth,
  Person,
  Description,  // 제거
  BarChart,      // 제거
  Lock,
  Search,        // 제거
  Notifications,
  Mail,
} from '@mui/icons-material';
```
- **무엇을 하는가**: 제거된 메뉴 아이콘(Description, BarChart, Search) import 삭제
- 사용하지 않는 import는 빌드 경고 발생 원인이 됨
