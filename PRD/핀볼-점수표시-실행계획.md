# 핀볼 점수 표시 실행계획

## 요구사항 요약

**요구사항**: Pinball.jsx 우상단에 현재 점수를 표시하는 텍스트를 추가

**목적**: 사용자의 동기부여를 위해 실시간으로 점수를 보여줄 필요가 있다

## 현재상태 분석

- Pinball.jsx는 Matter.js를 사용한 핀볼 게임 컴포넌트
- 700x1200 크기의 캔버스에 배경이미지가 적용되어 있음
- 현재 점수를 저장하고 표시하는 상태(state)가 없음
- 우상단에 점수를 표시할 UI 요소가 없음

## 구현 방법

**기술 스택**:
- React useState 훅으로 점수 상태 관리
- Material-UI Typography 컴포넌트로 점수 텍스트 표시

**UI 배치**:
- position: absolute로 게임 캔버스 우상단에 고정
- 배경색과 대비되는 색상으로 가독성 확보
- 모바일 환경을 고려한 크기 조정

## 구현 단계

### 1. 점수 상태 추가
- useState 훅으로 score 상태 변수 생성
- 초기값 0으로 설정

### 2. 점수 표시 UI 구현
- Box 컴포넌트로 점수 표시 영역 생성
- position: absolute, top: 20px, right: 20px로 우상단 배치
- Typography 컴포넌트로 "SCORE: {score}" 텍스트 표시
- 글자 크기 24px, 굵게(bold), 노란색(#FFD700) 적용
- text-shadow로 가독성 향상

### 3. 게임 캔버스와 점수 UI 레이아웃 조정
- 게임 캔버스를 감싸는 Box를 position: relative로 설정
- 점수 UI가 캔버스 내부 우상단에 위치하도록 구조 조정

### 4. Typography import 추가
- Material-UI에서 Typography 컴포넌트 import
