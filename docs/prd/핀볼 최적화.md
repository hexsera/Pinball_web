# 목표
현재 Pinball.jsx 가 모바일환경에서 느린 문제가 있다.
왜 느린지 조사하고 해당 문제를 해결할 방안을 여러개 제시하여라

## 역활
너는 10년차 프로프로그래머이다. 분석하고 해결법을 낼때는 전문가의 입장에서 생각해야하지만 설명할때에는 초보자에게 설명하듯 해야한다.

## 상세사항
1. 해결할 방안을 여러 방면에서 제시하라
2. 해결 방안을 제시할때 문제점이 무엇인지, 왜 해당 현상이 일어난지에 대해 초보자도 이해할 수 있게 각 10줄 내외로 작성한다. 해결방법에 대해서는 3줄 내외로 작성한다.
3. 분석내용을 md 파일로 작성해 prd 폴더에 만든다.
