# 목표
Hompage 헤더에 바로가기 섹션을 만든다.
현재 바로가기에 공지사항 (notice) 를 추가한다.
바로가기 섹션에는 현재 공지사항만 있지만 추후 아이템이 추가될 수 있음으로 고려해야한다.

바로가기 섹션 아이템 위에 마우스를 올리면 hover 효과가 나타나야한다.
아이템을 클릭하면 해당 아이템에 toggle 한 효과가 나타야한다.

Homepage 헤더를 frontend/src/pages/Notice/NoticeListPage.jsx 와 frontend/src/pages/Notice/NoticeDetailPage.jsx 에 추가한다. 이미 헤더가 있었다면 교체한다.
