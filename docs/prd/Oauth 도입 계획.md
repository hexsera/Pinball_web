# 목표
1. Oauth 2.0 을 통한 구글 로그인을 지원한다.
2. 내 웹사이트의 로그인 또한 Authorization Code 교환, 확인 후 JWT 토큰 지급을 구현한다.

# 상세사항
1. 프론트 엔드를 만드는 것을 PAHSE 1, 백엔드를 만드는 것을 PAHSE 2 로 구분한다.
2. JWT 이외에 라이브러리는 아직 사용하지 않는다.
3. Authorization Code 저장은 현재 api 서버 메모리에 저장한다.
4. 현재 로그인 세션 정보를 localstorge 에 저장해놓은 방식이다. 해당 방식을 바로 교체하지 않고, 현재는 혼용해서 사용하도록 하고 JWT 인증방식은 당장은 로그인시에만 사용하도록 한다.
