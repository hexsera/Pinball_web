# 목표
핀볼 게임에서 목숨이 모두 닳으면 게임오버가 되어 게임오버 메시지와 획득한 점수, 최고점수, 다시시작 버튼 UI 를 표시한다.

## 세부사항
1. mui box 로 핀볼게임 위를 덮는 UI 영역을 만든다.
2. 핀볼게임에서 목숨이 0 이 됐는지 확인한다.
3. 목숨이 0 이 됐다면 핀볼게임 위 UI 에서 게임오버 메시지와 획득한 점수, 최고 점수, 다시시작 버튼 UI 를 생성한다.
4. 최고 점수 는 api 로 get 을 해온다. (api 는 구현되어있지 않음)
5. 획득한 점수는 월간 랭킹 api "/api/v1/monthly-scores" 에 POST 한다.

## 주의사항
api 구현과 다시시작 버튼 작동은 현재 구현하지 않는다.
모든 작업을 한번에가 아니라 나눠서 작업하도록 PHASE 를 나눈다.
