# 공지사항 게시판 프론트엔드 실행계획

## 요구사항 요약

**요구사항**: 공지사항 목록/상세/작성 페이지를 React로 구현하고, App.jsx에 라우트를 등록하며 홈 페이지에 공지사항 링크를 추가한다.

**목적**: 관리자가 WYSIWYG 에디터로 이미지 포함 공지사항을 작성하고, 일반 사용자 및 비로그인 사용자가 목록과 상세 내용을 조회할 수 있도록 한다.

## 현재상태 분석

- `frontend/src/pages/` 하위에 각 페이지가 폴더 단위로 구성됨 (예: `HomePage/HomePage.jsx`)
- `App.jsx`에 react-router-dom `<Routes>`로 라우트 관리 중. 공지사항 라우트 없음.
- `AuthContext`에서 `user.role`로 admin 판별 가능. localStorage 기반.
- HomePage는 MUI `AppBar + Container + Grid` 레이아웃 사용. COLORS 상수로 색상 관리.
- 공지사항 관련 페이지, 컴포넌트, API 호출 코드 전혀 없음.

## 구현 방법

- Tiptap v3 에디터를 `NoticeWritePage`에 적용해 HTML 콘텐츠를 생성하고, 이를 API로 저장한다.
- 상세 페이지에서는 `dangerouslySetInnerHTML`로 저장된 HTML을 렌더링한다.
- API 호출은 `noticeService.js` 서비스 레이어로 분리한다. 페이지 컴포넌트에서 axios를 직접 호출하지 않는다.
- 페이지 레이아웃은 기존 HomePage의 COLORS 상수와 MUI 컴포넌트 스타일을 따른다.

## 구현 단계

### 1. Tiptap 패키지 설치

```bash
source ~/.nvm/nvm.sh && npm install @tiptap/react@3.20.1 @tiptap/starter-kit@3.20.1 @tiptap/extension-image@3.20.1
```

- `frontend/` 디렉토리에서 실행
- `@tiptap/react`: React용 에디터 바인딩
- `@tiptap/starter-kit`: Bold, Italic, Heading, Paragraph 등 기본 서식 확장 일괄 제공
- `@tiptap/extension-image`: 에디터 내 이미지 삽입 기능 추가

### 2. 서비스 레이어 생성 (`noticeService.js`)

```js
// frontend/src/services/noticeService.js
import axios from 'axios';

export const getNotices = () =>
  axios.get('/api/v1/notices').then(res => res.data);

export const getNotice = (id) =>
  axios.get(`/api/v1/notices/${id}`).then(res => res.data);

export const createNotice = (title, content) =>
  axios.post('/api/v1/notices', { title, content }).then(res => res.data);

export const deleteNotice = (id) =>
  axios.delete(`/api/v1/notices/${id}`).then(res => res.data);

export const uploadNoticeImage = (file) => {
  const formData = new FormData();
  formData.append('file', file);
  return axios.post('/api/v1/notices/upload-image', formData).then(res => res.data);
};
```

- 모든 공지사항 관련 API 호출을 한 곳에 모은다.
- 각 함수는 응답 데이터(`res.data`)만 반환해 페이지 컴포넌트가 axios를 직접 알 필요가 없게 한다.

### 3. 공지사항 목록 페이지 생성 (`NoticeListPage.jsx`)


```jsx
// frontend/src/pages/Notice/NoticeListPage.jsx
import { useState, useEffect, useContext } from 'react';
import { useNavigate } from 'react-router-dom';
import { Box, AppBar, Toolbar, Typography, Button, Container, List,
         ListItem, ListItemText, Divider, Paper } from '@mui/material';
import { AuthContext } from '../../contexts/AuthContext';
import HeaderUserInfo from '../../components/HeaderUserInfo';
import { getNotices } from '../../services/noticeService';

const COLORS = { bg: '#0F172A', card: '#1E293B', border: '#334155',
                 text: '#F1F5F9', subText: '#94A3B8', primary: '#4F46E5' };

function NoticeListPage() {
  const navigate = useNavigate();
  const { user } = useContext(AuthContext);
  const [notices, setNotices] = useState([]);

  useEffect(() => {
    getNotices().then(setNotices);
  }, []);

  return (
    <Box sx={{ minHeight: '100vh', backgroundColor: COLORS.bg }}>
      <AppBar position="static" sx={{ backgroundColor: COLORS.card,
                                      borderBottom: `1px solid ${COLORS.border}` }}>
        <Toolbar sx={{ justifyContent: 'space-between' }}>
          <Typography variant="h6" sx={{ color: COLORS.text, cursor: 'pointer' }}
                      onClick={() => navigate('/')}>HEXSERA PINBALL</Typography>
          <HeaderUserInfo buttonColor={COLORS.primary} buttonTextColor={COLORS.text}
                          outlinedBorderColor={COLORS.text} />
        </Toolbar>
      </AppBar>
      <Container maxWidth="md" sx={{ py: 4 }}>
        <Box sx={{ display: 'flex', justifyContent: 'space-between', mb: 2 }}>
          <Typography variant="h5" sx={{ color: COLORS.text }}>공지사항</Typography>
          {user?.role === 'admin' && (
            <Button variant="contained" onClick={() => navigate('/notice/write')}
                    sx={{ backgroundColor: COLORS.primary }}>글쓰기</Button>
          )}
        </Box>
        <Paper sx={{ backgroundColor: COLORS.card, border: `1px solid ${COLORS.border}` }}>
          <List>
            {notices.map((notice, idx) => (
              <Box key={notice.id}>
                <ListItem button onClick={() => navigate(`/notice/${notice.id}`)}>
                  <ListItemText
                    primary={<Typography sx={{ color: COLORS.text }}>{notice.title}</Typography>}
                    secondary={<Typography sx={{ color: COLORS.subText, fontSize: '0.8rem' }}>
                      {new Date(notice.created_at).toLocaleDateString('ko-KR')}
                    </Typography>}
                  />
                </ListItem>
                {idx < notices.length - 1 && <Divider sx={{ borderColor: COLORS.border }} />}
              </Box>
            ))}
          </List>
        </Paper>
      </Container>
    </Box>
  );
}

export default NoticeListPage;
```

- `user?.role === 'admin'`일 때만 글쓰기 버튼 노출
- `GET /api/v1/notices` 호출로 목록 조회
- 각 항목 클릭 시 `/notice/{id}` 로 이동

### 4. 공지사항 작성 페이지 생성 (`NoticeWritePage.jsx`)

```jsx
// frontend/src/pages/Notice/NoticeWritePage.jsx
import { useState, useContext, useEffect } from 'react';
import { useNavigate } from 'react-router-dom';
import { useEditor, EditorContent } from '@tiptap/react';
import StarterKit from '@tiptap/starter-kit';
import Image from '@tiptap/extension-image';
import { Box, AppBar, Toolbar, Typography, Button, TextField,
         Container, Paper } from '@mui/material';
import { AuthContext } from '../../contexts/AuthContext';
import { createNotice, uploadNoticeImage } from '../../services/noticeService';

const COLORS = { bg: '#0F172A', card: '#1E293B', border: '#334155',
                 text: '#F1F5F9', subText: '#94A3B8', primary: '#4F46E5' };

function NoticeWritePage() {
  const navigate = useNavigate();
  const { user } = useContext(AuthContext);
  const [title, setTitle] = useState('');

  // admin이 아니면 접근 차단
  useEffect(() => {
    if (!user || user.role !== 'admin') navigate('/notice');
  }, [user]);

  const editor = useEditor({
    extensions: [StarterKit, Image],
    content: '',
  });

  const handleImageUpload = async (e) => {
    const file = e.target.files[0];
    if (!file) return;
    const data = await uploadNoticeImage(file);
    editor.chain().focus().setImage({ src: data.url }).run();
  };

  const handleSubmit = async () => {
    const content = editor.getHTML();
    await createNotice(title, content);
    navigate('/notice');
  };

  return (
    <Box sx={{ minHeight: '100vh', backgroundColor: COLORS.bg }}>
      <AppBar position="static" sx={{ backgroundColor: COLORS.card,
                                      borderBottom: `1px solid ${COLORS.border}` }}>
        <Toolbar>
          <Typography variant="h6" sx={{ color: COLORS.text, cursor: 'pointer' }}
                      onClick={() => navigate('/')}>HEXSERA PINBALL</Typography>
        </Toolbar>
      </AppBar>
      <Container maxWidth="md" sx={{ py: 4 }}>
        <Typography variant="h5" sx={{ color: COLORS.text, mb: 2 }}>공지사항 작성</Typography>
        <TextField fullWidth label="제목" value={title}
                   onChange={e => setTitle(e.target.value)}
                   sx={{ mb: 2, input: { color: COLORS.text },
                         label: { color: COLORS.subText },
                         '& .MuiOutlinedInput-root': { '& fieldset': { borderColor: COLORS.border } } }} />
        <Paper sx={{ backgroundColor: '#fff', p: 2, mb: 2, minHeight: 300 }}>
          {/* 에디터 툴바 */}
          <Box sx={{ mb: 1, display: 'flex', gap: 1, flexWrap: 'wrap' }}>
            <Button size="small" variant="outlined"
                    onClick={() => editor.chain().focus().toggleBold().run()}>B</Button>
            <Button size="small" variant="outlined"
                    onClick={() => editor.chain().focus().toggleItalic().run()}>I</Button>
            <Button size="small" variant="outlined"
                    onClick={() => editor.chain().focus().toggleHeading({ level: 2 }).run()}>H2</Button>
            <Button size="small" variant="outlined" component="label">
              이미지 <input type="file" hidden accept="image/*" onChange={handleImageUpload} />
            </Button>
          </Box>
          <EditorContent editor={editor} />
        </Paper>
        <Box sx={{ display: 'flex', gap: 1, justifyContent: 'flex-end' }}>
          <Button variant="outlined" onClick={() => navigate('/notice')}
                  sx={{ color: COLORS.subText, borderColor: COLORS.border }}>취소</Button>
          <Button variant="contained" onClick={handleSubmit}
                  sx={{ backgroundColor: COLORS.primary }}>저장</Button>
        </Box>
      </Container>
    </Box>
  );
}

export default NoticeWritePage;
```

- `useEffect`에서 admin이 아닌 경우 `/notice`로 리다이렉트해 페이지 접근 자체를 차단
- `useEditor`로 Tiptap 에디터 인스턴스 생성. `StarterKit`과 `Image` 확장 등록.
- 이미지 업로드: `FormData`로 파일 전송 후 반환된 URL을 `setImage`로 에디터에 삽입
- 저장 시 `editor.getHTML()`로 HTML 추출해 API로 전송

### 5. 공지사항 상세 페이지 생성 (`NoticeDetailPage.jsx`)

```jsx
// frontend/src/pages/Notice/NoticeDetailPage.jsx
import { useState, useEffect, useContext } from 'react';
import { useParams, useNavigate } from 'react-router-dom';
import { Box, AppBar, Toolbar, Typography, Button, Container,
         Paper, Divider } from '@mui/material';
import { AuthContext } from '../../contexts/AuthContext';
import HeaderUserInfo from '../../components/HeaderUserInfo';
import { getNotice, deleteNotice } from '../../services/noticeService';

const COLORS = { bg: '#0F172A', card: '#1E293B', border: '#334155',
                 text: '#F1F5F9', subText: '#94A3B8', primary: '#4F46E5' };

function NoticeDetailPage() {
  const { id } = useParams();
  const navigate = useNavigate();
  const { user } = useContext(AuthContext);
  const [notice, setNotice] = useState(null);

  useEffect(() => {
    getNotice(id).then(setNotice);
  }, [id]);

  const handleDelete = async () => {
    await deleteNotice(id);
    navigate('/notice');
  };

  if (!notice) return null;

  return (
    <Box sx={{ minHeight: '100vh', backgroundColor: COLORS.bg }}>
      <AppBar position="static" sx={{ backgroundColor: COLORS.card,
                                      borderBottom: `1px solid ${COLORS.border}` }}>
        <Toolbar sx={{ justifyContent: 'space-between' }}>
          <Typography variant="h6" sx={{ color: COLORS.text, cursor: 'pointer' }}
                      onClick={() => navigate('/')}>HEXSERA PINBALL</Typography>
          <HeaderUserInfo buttonColor={COLORS.primary} buttonTextColor={COLORS.text}
                          outlinedBorderColor={COLORS.text} />
        </Toolbar>
      </AppBar>
      <Container maxWidth="md" sx={{ py: 4 }}>
        <Button onClick={() => navigate('/notice')}
                sx={{ color: COLORS.subText, mb: 2 }}>← 목록으로</Button>
        <Paper sx={{ backgroundColor: COLORS.card, border: `1px solid ${COLORS.border}`, p: 3 }}>
          <Typography variant="h5" sx={{ color: COLORS.text, mb: 1 }}>{notice.title}</Typography>
          <Typography sx={{ color: COLORS.subText, fontSize: '0.85rem', mb: 2 }}>
            {new Date(notice.created_at).toLocaleDateString('ko-KR')}
          </Typography>
          <Divider sx={{ borderColor: COLORS.border, mb: 2 }} />
          {/* 저장된 HTML 콘텐츠를 그대로 렌더링 */}
          <Box sx={{ color: COLORS.text, '& img': { maxWidth: '100%' } }}
               dangerouslySetInnerHTML={{ __html: notice.content }} />
          {user?.role === 'admin' && (
            <Box sx={{ display: 'flex', gap: 1, justifyContent: 'flex-end', mt: 3 }}>
              <Button variant="outlined"
                      onClick={() => navigate(`/notice/${id}/edit`)}
                      sx={{ color: COLORS.text, borderColor: COLORS.border }}>수정</Button>
              <Button variant="contained" color="error" onClick={handleDelete}>삭제</Button>
            </Box>
          )}
        </Paper>
      </Container>
    </Box>
  );
}

export default NoticeDetailPage;
```

- `useParams()`로 URL의 `:id` 추출 후 `GET /api/v1/notices/{id}` 호출
- `dangerouslySetInnerHTML`로 서버에서 받은 HTML을 렌더링. `img`의 `maxWidth: 100%`로 이미지 오버플로 방지.
- admin에게만 수정/삭제 버튼 노출

```jsx
// frontend/src/App.jsx - 추가할 import
import NoticeListPage from './pages/Notice/NoticeListPage';
```
