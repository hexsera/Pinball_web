# 목표
UserInfo.jsx 에서 회원 탈퇴 버튼을 누르면 DialogTitle 컴포넌트에서 비밀번호 확인을 한 후 비밀번호가 같다면 해당 회원을 삭제시키고 AuthContext 를 비운 후 메인 페이지로 보낸다.

## 상세내용
1. DialogTile 에는 "정말 탈되하시겠습니까? 삭제된 정보는 되돌릴 수 없습니다" "진행하려면 비밀번호를 입력해 주세요" 를 표시하고 Textinput 으로 비밀번호를 적도록 하고, 확인, 취소 버튼을 배치한다.
2. 확인 버튼을 누르면 api 의 login 을 통해 해당 비밀번호가 맞는지 확인한다.
3. 비밀번호가 맞다면 api/v1/users 에 DELETE 로 유저 삭제 요청을 보낸다.
4. 그후 AuthContext 에서 로그인 정보를 지우고 메인페이지로 보낸다.
