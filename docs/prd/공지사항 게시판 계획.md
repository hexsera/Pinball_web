# 목표
웹사이트에 공지사항 계시판을 생성하도록 한다.

공지사항 목록 페이지
1. 공지사항 목록 페이지는 홈 페이지에서 들어갈 수 있도록 한다.
2. 공지사항 목록 페이지에 들어가면 등록된 공지사항 제목을 볼 수 있어야한다.
3. 공지사항 목록 페이지에 어드민계정으로 접속하면 글 작성 버튼을 볼 수 있어야한다.
4. 공지사항 제목을 누르면 해당 공지사항 페이지로 넘어간다.

공지사항 작성 페이지
1. 공지사항 작성 페이지는 글쓰기 기능이 있어야한다.
2. 글쓰기 기능은 WYSIWYG 하도록 해야한다.
3. 이미지 첨부가 가능해야한다. 이미지는 글자 사이에 넣을 수 있어야한다.

공지사항 페이지
1. 공지사항 페이지에 들어가면 작성된 공지사항 페이지를 볼 수 있어야한다.
