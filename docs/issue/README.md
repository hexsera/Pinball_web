# docs/issue/

작업 중 발생한 미해결 이슈를 기록하는 폴더다.

## 용도

- 작업 도중 발견했지만 즉시 해결하지 못한 문제
- 추후 검토가 필요한 버그, 설계 문제, 기술 부채
- 현재 진행 중인 작업과 직접 관련 없어 보류한 사항

## AI를 위한 안내

이 폴더에 문서를 저장할 때:
- 파일명: `YYYY-MM-DD-간단한-제목.md` 형식 사용
- 이슈가 해결되면 해당 파일을 삭제하거나 `resolved` 표시 후 `docs/history/`로 이동
- 하나의 파일에 하나의 이슈만 기록
- 미해결 상태로 남겨야 할 이슈만 이 폴더에 보관
