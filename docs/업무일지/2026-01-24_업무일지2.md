## 주요 변경 사항

### 1. react/main/src/Pinball.jsx 수정

#### (1) Matter.js Render 옵션 수정 (58행)

**변경 전:**
```javascript
    const render = Render.create({
      element: sceneRef.current,
      engine: engine,
      options: {
        width: 700,
        height: 1200,
        wireframes: false,
        background: '/images/pinball_back.png'
      }
    });
```

**변경 후:**
```javascript
    const render = Render.create({
      element: sceneRef.current,
      engine: engine,
      options: {
        width: 700,
        height: 1200,
        wireframes: false,
        background: 'transparent'
      }
    });
```

**특징:**
- Matter.js 캔버스의 배경을 투명하게 설정
- 배경이미지는 Box 컴포넌트에서 CSS로 처리

#### (2) Box 컴포넌트에 배경이미지 스타일 적용 (343-352행)

**변경 전:**
```javascript
        <Box sx={{
          transform: { xs: 'scale(0.5)', sm: 'scale(0.5)', md: 'scale(0.8)' },
          transformOrigin: 'top center'
        }}>
          <div ref={sceneRef} />
        </Box>
```

**변경 후:**
```javascript
        <Box sx={{
          width: '700px',
          height: '1200px',
          backgroundImage: 'url(/images/pinball_back.png)',
          backgroundSize: '100% 100%',
          backgroundPosition: 'center',
          backgroundRepeat: 'no-repeat',
          transform: { xs: 'scale(0.5)', sm: 'scale(0.5)', md: 'scale(0.8)' },
          transformOrigin: 'top center'
        }}>
          <div ref={sceneRef} />
        </Box>
```

**특징:**
- Box 크기를 캔버스 크기(700×1200)와 동일하게 지정
- backgroundImage로 배경이미지 경로 설정
- backgroundSize: '100% 100%'로 이미지를 Box 크기에 정확히 맞춤
- backgroundPosition: 'center'로 중앙 정렬
- backgroundRepeat: 'no-repeat'로 이미지 반복 방지
- 기존 transform, transformOrigin은 유지
