# 목적
내 웹사이트에 구글 로그인을 도입한다. 단, 실제 DB 에 유저 추가는 아직 구현하지 않는다.
프론트엔드에 react-oauth/google 라이브러리, 백엔드 Authlib 라이브러리를 이용해 Authorization Code Flow 을 하도록 한다.
프론트 엔드 Login.jsx 에 회원가입 버튼 위에 구글 로그인 모양 버튼 만들고 해당 버튼을 누르면 구글 로그인을 하도록 한다.

## 상세사항
1. Oauth 연결에 필요한 변수값들을 나에게 알려줘서 준비할 수 있겠끔 하여라. (준비 가이드는 필요없음)
2. 코드에 기능별로 주석을 남겨 무엇을 하는지 알 수 있게 하여라.
