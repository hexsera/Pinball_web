# docs/history

작업 진행 기록을 관리하는 디렉토리입니다.

## 파일 명명 규칙

```
YYYY-MM-DD-[주제ID]-순번-제목.md
```

- **주제 ID**: 주제를 처음 시작한 날 기준으로 `sha256("YYYY-MM-DD-주제슬러그")` 앞 6자리
- **순번**: 같은 날 같은 주제 내 순서 (01, 02, ...)
- 다른 날 같은 주제라면 최초 생성한 주제 ID를 재사용

### 주제 ID 생성

```bash
echo -n "2026-04-06-oauth" | sha256sum | head -c 6
```


### 예시

```
2026-04-06-a3f7c2-01-oauth구현-초기설계.md
2026-04-06-9b1e4d-01-api패키지설치.md
2026-04-07-a3f7c2-02-oauth구현-토큰처리.md
```

## 작성 가이드

같은 주제 ID를 가진 이전/이후 파일이 있다면 문서 상단에 생성 순서대로 링크를 첨부한다.

내용을 추가할 때는 사용자의 지시가 없으면 맨 아래 또는 맥락적으로 자연스러운 위치에 넣는다.

```markdown
**관련 파일 (주제 ID: a3f7c2):**
- [2026-04-06-a3f7c2-01-oauth구현-초기설계.md](2026-04-06-a3f7c2-01-oauth구현-초기설계.md)
- [2026-04-07-a3f7c2-02-oauth구현-토큰처리.md](2026-04-07-a3f7c2-02-oauth구현-토큰처리.md)
```
