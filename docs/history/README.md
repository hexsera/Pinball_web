# History

이 폴더는 Claude와의 대화를 통해 진행된 작업 기록을 관리합니다.

## 목적

Claude와 대화하며 논의하고 토의한 내용과 결정사항의 **맥락**을 추적한다.

- 업무일지(실제 진행된 업무의 변경점 정리)와 다름
- 어떤 주제를 논의했고, 어떤 이유로 무엇을 결정했는지를 기록
- 이후 대화에서 같은 맥락을 빠르게 복원하기 위한 참고 자료

## 파일 명명 규칙

```
YYYY-MM-DD-순번-문서제목.md
```

예시:
- `2026-03-31-01-세션-유지-구현.md`
- `2026-03-31-02-회원가입-수정.md`
