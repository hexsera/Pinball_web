## 현상

`npm run build` 실행 시 아래 경고 발생:

```
(!) Some chunks are larger than 500 kB after minification.
dist/assets/index-C2pX87eO.js   1,892.09 kB │ gzip: 582.73 kB
```

## 원인

모든 코드(Matter.js 물리 엔진 포함)가 JS 파일 하나에 묶여 있어 번들 크기가 1.9MB에 달함.

## 영향

첫 접속 시 1.9MB JS 파일을 한 번에 내려받아야 함. 이후 접속은 브라우저 캐시로 빠름.

## 해결 방법 (나중에)

dynamic import로 코드 스플리팅 적용. 핀볼 페이지 진입 시에만 Matter.js 로드하는 방식.

```javascript
// 예시: 라우터에서 Pinball 페이지 lazy load
const Pinball = lazy(() => import('./pages/Pinball/Pinball'));
```
