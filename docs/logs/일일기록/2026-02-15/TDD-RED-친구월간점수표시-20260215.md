# TDD RED 계획: 현재 친구 목록에 월간 최고점수 표시

## 구현 목표
현재 친구 목록에 각 친구의 월간 최고점수를 함께 표시한다.
- 친구 목록 로드 후 각 친구의 user_id로 GET /api/v1/monthly-scores/{user_id} 호출
- 점수가 있으면 숫자 표시, 없으면 "-" 표시

## RED 테스트 목록

1. `친구 목록 로드 시 각 친구의 GET /api/v1/monthly-scores/{user_id}를 호출한다`: 친구 2명이면 monthly-scores API 2번 호출
2. `친구의 월간 점수가 있으면 점수가 표시된다`: API 응답의 score 값이 화면에 표시됨
3. `친구의 월간 점수가 없으면 "-"가 표시된다`: 404 응답 시 "-" 표시
