# 로그인 배경 변경 실행계획

## 요구사항 요약

**요구사항**: Login.jsx의 색상 테마를 변경하고, `#0F172A` 기반 배경에 `#467ee5`, `#7C3AED`, `#908aff` 포인트 색상으로 Aurora 애니메이션 그라데이션 배경을 적용한다.

**목적**: 로그인 페이지의 시각적 완성도를 높이고 HomePage와 통일된 디자인 언어를 제공한다.

## 현재상태 분석

- `Login.jsx`: 배경색 `#87CEEB`(하늘색), Paper 컴포넌트는 흰색(`white`), 버튼은 MUI 기본 primary 색상 사용
- `HomePage.jsx`: `#0F172A` 배경, Aurora 컴포넌트(`#467ee5`, `#7C3AED`, `#908aff`)로 애니메이션 배경 적용, `COLORS` 객체로 색상 통합 관리
- `Aurora.jsx`: WebGL 기반 노이즈 애니메이션 컴포넌트로 재사용 가능한 구조
- Login.jsx는 Aurora 컴포넌트를 import하지 않아 배경 효과가 없는 상태

## 구현 방법

- HomePage.jsx와 동일한 구조(`Aurora` + z-index 레이어 구조)를 Login.jsx에 적용
- `COLORS` 객체로 색상을 통합 관리하여 일관성 유지
- Paper(카드) 배경은 반투명 dark 색상으로 변경하여 Aurora 배경이 비치도록 처리

## 구현 단계

### 1. Aurora import 및 COLORS 정의 추가

```jsx
import Aurora from '../../components/Aurora/Aurora';

const COLORS = {
  bg: '#0F172A',
  card: 'rgba(30, 41, 59, 0.85)',
  border: '#334155',
  text: '#F1F5F9',
  subText: '#94A3B8',
  primary: '#4F46E5',
};
```
- **무엇을 하는가**: HomePage와 동일한 색상 팔레트와 Aurora 컴포넌트를 Login에서 사용할 수 있도록 준비
- `card`는 `rgba`로 반투명 처리해 Aurora 배경이 Paper 뒤로 비치도록 설정
- `text`는 어두운 배경에서 가독성을 위해 밝은 색상 사용

### 2. 최외곽 Box에 Aurora 레이어 구조 적용

```jsx
<Box sx={{ position: 'relative', minHeight: '100vh', backgroundColor: COLORS.bg, display: 'flex', alignItems: 'center', justifyContent: 'center' }}>
  {/* Aurora 배경 */}
  <Box sx={{ position: 'absolute', inset: 0, zIndex: 0, overflow: 'hidden' }}>
    <Aurora
      colorStops={['#467ee5', '#7C3AED', '#908aff']}
      amplitude={1.2}
      speed={1.2}
      blend={0.35}
    />
  </Box>

  {/* 기존 콘텐츠 */}
  <Box sx={{ position: 'relative', zIndex: 1, width: '100%', display: 'flex', alignItems: 'center', justifyContent: 'center' }}>
    {/* Container + Paper ... */}
  </Box>
</Box>
```
- **무엇을 하는가**: Aurora 레이어(z-index: 0)를 절대 위치로 전체 화면에 깔고, 로그인 폼(z-index: 1)을 그 위에 배치
- `inset: 0`은 상/하/좌/우를 모두 0으로 설정해 부모를 꽉 채움
- `colorStops`, `amplitude`, `speed`, `blend`는 HomePage와 동일한 값 사용

### 3. Paper 및 텍스트 색상 변경

```jsx
<Paper
  elevation={3}
  sx={{
    padding: 4,
    backgroundColor: COLORS.card,
    border: `1px solid ${COLORS.border}`,
    borderRadius: 2,
  }}
>
  <Typography component="h1" variant="h5" sx={{ color: COLORS.text }}>
    로그인
  </Typography>
  <Typography component="h6" variant="h6" sx={{ color: COLORS.text }}>
    이메일
  </Typography>
  <Typography component="h6" variant="h6" sx={{ color: COLORS.text }}>
    비밀번호
  </Typography>
```
- **무엇을 하는가**: Paper 카드 배경을 반투명 dark 색상으로, 텍스트를 밝은 색상으로 변경해 어두운 배경에서 가독성 확보
- `border`를 추가해 카드 경계를 시각적으로 구분

### 4. TextField 스타일 변경

```jsx
<TextField
  fullWidth
  sx={{
    mt: 1,
    '& .MuiOutlinedInput-root': {
      color: COLORS.text,
      '& fieldset': { borderColor: COLORS.border },
      '&:hover fieldset': { borderColor: COLORS.text },
      '&.Mui-focused fieldset': { borderColor: COLORS.primary },
    },
    '& .MuiInputBase-input::placeholder': { color: COLORS.subText },
  }}
  // ... 나머지 props 동일
/>
```
- **무엇을 하는가**: TextField 테두리·입력 텍스트·placeholder 색상을 dark 테마에 맞게 변경
- MUI의 `sx` prop에서 CSS-in-JS 방식으로 내부 하위 요소(fieldset, input)를 직접 스타일링

### 5. 버튼 색상 변경

```jsx
<Button
  variant="contained"
  sx={{ width: '300px', backgroundColor: COLORS.primary, color: COLORS.text }}
  onClick={handleLogin}
>
  로그인
</Button>
<Button
  variant="outlined"
  sx={{ width: '300px', borderColor: COLORS.text, color: COLORS.text }}
  onClick={() => navigate('/Register')}
>
  회원가입
</Button>
```
- **무엇을 하는가**: 로그인 버튼은 `primary` 색상(`#4F46E5`)으로, 회원가입 버튼은 밝은 테두리·텍스트 색상으로 변경
- HomePage의 버튼 스타일과 동일한 패턴 적용
