# TDD GREEN 완료: 친구페이지 요청목록 및 친구목록 표시 수정

## 구현 완료 항목
- 친구 요청란: receiver_id가 내 id인 요청만 필터링하여 표시
- 친구 요청란: requester_nickname 표시 (기존 "요청 ID: N" → 요청자 닉네임)
- 친구 요청란: 승인 버튼 color를 success(초록색)로 변경
- 현재 친구란: receiver_id === 내 id이면 requester_nickname, requester_id === 내 id이면 receiver_nickname 표시 (기존 "친구 ID: N" → 상대방 닉네임)
- 사용하지 않는 getFriendId 함수 제거

## 통과한 테스트 목록 (신규 4개)
1. receiver_id가 내 id인 요청만 친구 요청란에 표시된다
2. 친구 요청란에 requester_nickname이 표시된다
3. receiver_id가 내 id인 경우 requester_nickname을 친구 닉네임으로 표시한다
4. requester_id가 내 id인 경우 receiver_nickname을 친구 닉네임으로 표시한다
