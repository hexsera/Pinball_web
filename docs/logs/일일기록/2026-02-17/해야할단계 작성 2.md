# 목표
핀볼 게임에 멀티플레이를 도입하려고 한다.

## 상세사항
1. 현재 핀볼게임은 싱글플레이 시나리오이다. 멀티 플레이는 멀티플레이 시나리오를 가져야한다.
2. 멀티플레이는 상대 플립퍼가 있다.
3. "멀티플레이 시작" 버튼을 누르면 멀티플레이 시나리오를 시작하도록한다.

## 주의사항
1. api 와 보안에 대해 현재는 구현하지않고 매우 간단한 수준만 시행하도록한다.
2. 멀티플레이 시나리오는 매우 간단하게, 내 플립퍼, 상대 플립퍼, 공, 벽만 존재하도록 한다.
3. 기존 싱글플레이에서 사용 할 수 있는 객체는 사용한다.
4. 상세한 코드를 작성하지 않는다. 어떤걸 해야한다 와 같은 단계를 작성하고 설명은 3줄 정도 작성하도록 한다.
