# 핀볼 게임 사이트 페이지 레이아웃 설계

---

## 필요한 페이지 목록

1. **홈 (랜딩 페이지)** - 사이트 첫 진입 화면
2. **로그인 페이지** - 이메일/비밀번호 로그인
3. **회원가입 페이지** - 단계별 회원가입 폼
4. **게임 대시보드** - 게임 플레이 + 메뉴 통합 화면
5. **회원 정보 페이지** - 내 정보 조회/수정/탈퇴
6. **랭킹 페이지** - 전체 점수 랭킹 Top 10
7. **어드민 페이지** - 관리자 전용 유저 관리

---

## 1. 홈 (랜딩 페이지) `/`

Hero 섹션 좌측에 타이틀 + CTA 버튼, 우측에 월간 랭킹 테이블을 나란히 배치하는 2단 레이아웃.

```
┌──────────────────────────────────────────────────────────────────┐
│  LOGO    게임하기                [로그인]  [회원가입]               │  ← 상단 내비게이션 바
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
│   ┌──────────────────────────┐  ┌──────────────────────────────┐ │
│   │                          │  │  🏆 이번 달 랭킹               │ │
│   │   🎯 HEXSERA PINBALL     │  │                              │ │
│   │                          │  │  ┌───┬──────────┬──────────┐ │ │
│   │  세상에서 가장 짜릿한      │  │  │ # │  닉네임  │   점수   │ │ │
│   │  핀볼 게임을 경험하세요.   │  │  ├───┼──────────┼──────────┤ │ │  ← Hero 섹션 (2단)
│   │                          │  │  │🥇 │ alpha    │1,234,500 │ │ │
│   │                          │  │  │🥈 │ ninja    │  987,300 │ │ │
│   │  [▶ 지금 바로 플레이하기]  │  │  │🥉 │ hexfan   │  765,000 │ │ │
│   │                          │  │  │ 4 │ king     │  543,200 │ │ │
│   │  [📋 전체 랭킹 보기]       │  │  │ 5 │ 홍길동   │  321,000 │ │ │
│   │                          │  │  └───┴──────────┴──────────┘ │ │
│   └──────────────────────────┘  └──────────────────────────────┘ │
│                                                                  │
├──────────────────────────────────────────────────────────────────┤
│                       © 2026 HEXSERA                            │  ← 푸터
└──────────────────────────────────────────────────────────────────┘
```

**구성 요소**
- 네비게이션 바: 로고(좌), 게임하기 링크(중앙), 로그인·회원가입 버튼(우)
- Hero 섹션 (2단 분할):
  - 좌측: 사이트 타이틀 + 서브 문구 + **지금 바로 플레이하기** CTA 버튼 + 전체 랭킹 보기 버튼
  - 우측: **이번 달 랭킹 테이블** (순위·닉네임·점수, 상위 5명, 1~3위 메달 아이콘)
- 푸터: © 2026 HEXSERA

**반응형 처리**
- 데스크톱(md↑): 좌·우 2단 나란히 배치
- 모바일(sm↓): 타이틀 + CTA 버튼 → 월간 랭킹 테이블 순으로 단일 컬럼 스택

## 2. 로그인 페이지 `/login`

화면 중앙 카드 레이아웃. 배경은 어두운 핀볼 테마.

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│                                                     │
│              ┌───────────────────┐                  │
│              │  🎯  HEXSERA      │                  │
│              │                   │                  │
│              │  이메일            │                  │
│              │  ┌─────────────┐  │                  │
│              │  │             │  │                  │
│              │  └─────────────┘  │                  │
│              │                   │  ← 중앙 카드       │
│              │  비밀번호          │                  │
│              │  ┌─────────────┐  │                  │
│              │  │  ●●●●●●●   │  │                  │
│              │  └─────────────┘  │                  │
│              │                   │                  │
│              │  [   로그인하기  ]  │                  │
│              │                   │                  │
│              │  회원가입 | 비밀번호 찾기               │
│              └───────────────────┘                  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**구성 요소**
- 배경: 핀볼 테마 어두운 배경 (또는 블러 처리)
- 카드: 로고 + 이메일 + 비밀번호 + 로그인 버튼
- 하단 링크: 회원가입, 비밀번호 찾기

## 3. 회원가입 페이지 `/register`

단계 표시(Step Indicator)와 함께 중앙 카드 레이아웃.

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│         ①────────②────────③                        │  ← Step 표시
│        정보입력   비밀번호   생년월일                  │
│                                                     │
│              ┌───────────────────┐                  │
│              │                   │                  │
│              │  이메일            │                  │
│              │  ┌─────────────┐  │                  │
│              │  │             │  │                  │
│              │  └─────────────┘  │                  │
│              │                   │                  │  ← 현재 단계 폼
│              │  닉네임            │                  │
│              │  ┌─────────────┐  │                  │
│              │  │             │  │                  │
│              │  └─────────────┘  │                  │
│              │                   │                  │
│              │       [다음 →]    │                  │
│              └───────────────────┘                  │
│                                                     │
│                  이미 계정이 있으신가요? 로그인         │
└─────────────────────────────────────────────────────┘
```

**구성 요소**
- Step Indicator: 3단계 (정보 입력 → 비밀번호 → 생년월일)
- 카드: 현재 단계 폼
- 이전/다음 버튼
- 로그인 링크

## 4. 게임 대시보드 `/dashboard`

사이드바 + 메인 콘텐츠 2단 레이아웃. 핀볼 플레이가 중심.

```
┌────────────────────────────────────────────────────────────────┐
│ 🎯 HEXSERA                    👤 닉네임님   🔔  [로그아웃]       │  ← 상단 헤더
├──────────────┬─────────────────────────────────────────────────┤
│              │                                                 │
│  👤 계정     │                                                 │
│              │                                                 │
│  🎮 게임하기 │          [ 핀볼 게임 화면 ]                      │
│              │                                                 │
│  🏆 랭킹     │              (Pinball.jsx)                      │  ← 메인 콘텐츠
│              │                                                 │
│  📊 내 점수  │                                                 │
│              │                                                 │
│              │                                                 │
│  ──────────  │                                                 │
│  ⚙️ 설정    │                                                 │
│              │                                                 │
└──────────────┴─────────────────────────────────────────────────┘
  ← 사이드바 →   ← 메인 영역 (사이드바 클릭에 따라 콘텐츠 변경) →
```

**구성 요소**
- 헤더: 로고(좌), 닉네임·알림·로그아웃(우)
- 사이드바: 계정, 게임하기, 랭킹, 내 점수, 설정
- 메인 영역: 사이드바 선택에 따라 조건부 렌더링

## 5. 회원 정보 페이지 (대시보드 내 렌더링)

대시보드 메인 영역에 표시되는 카드 3개 레이아웃.

```
┌─────────────────────────────────────────────────────┐
│  내 정보                                             │
│ ┌─────────────────────────────────────────────────┐ │
│ │  📧 이메일     user@example.com                  │ │  ← 정보 조회 카드
│ │  👤 닉네임     홍길동                              │ │
│ │  🎂 생년월일   2000-01-01                         │ │
│ └─────────────────────────────────────────────────┘ │
│                                                     │
│  정보 수정                                           │
│ ┌─────────────────────────────────────────────────┐ │
│ │  닉네임  [ 홍길동          ]                       │ │  ← 수정 카드
│ │  비밀번호 [ ●●●●●●●●●   ]                       │ │
│ │                          [저장하기]               │ │
│ └─────────────────────────────────────────────────┘ │
│                                                     │
│  회원 탈퇴                                           │
│ ┌─────────────────────────────────────────────────┐ │
│ │  탈퇴 시 모든 데이터가 삭제됩니다.                   │ │  ← 탈퇴 카드
│ │                         [회원 탈퇴]               │ │
│ └─────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

**구성 요소**
- 정보 조회 카드: 이메일, 닉네임, 생년월일 표시
- 정보 수정 카드: 닉네임, 비밀번호 수정 폼
- 회원 탈퇴 카드: 경고 문구 + 탈퇴 버튼 (빨간색)

## 6. 랭킹 페이지 (대시보드 내 렌더링)

테이블 형태의 랭킹 리스트. 상위 3명은 아이콘으로 강조.

```
┌─────────────────────────────────────────────────────┐
│  🏆 전체 랭킹                                        │
│                                                     │
│  ┌───┬────────────────┬───────────────────────────┐ │
│  │ # │    닉네임       │          점수              │ │
│  ├───┼────────────────┼───────────────────────────┤ │
│  │ 🥇│ player_alpha   │       1,234,500           │ │
│  │ 🥈│ ninja_ball     │         987,300           │ │
│  │ 🥉│ hexsera_fan    │         765,000           │ │
│  │  4│ pinball_king   │         543,200           │ │
│  │  5│ 홍길동          │         321,000           │ │
│  │  6│ ...            │             ...           │ │
│  └───┴────────────────┴───────────────────────────┘ │
│                                                     │
│  📍 내 순위:  12위  |  점수: 128,000               │  ← 내 순위 강조
└─────────────────────────────────────────────────────┘
```

**구성 요소**
- 타이틀: 전체 랭킹 헤더
- 랭킹 테이블: 순위, 닉네임, 점수 (상위 3명 아이콘 강조)
- 내 순위 배너: 로그인 시 내 순위 하단 표시

## 7. 어드민 페이지 `/admin`

관리자 전용. 사이드바 + 유저 관리 테이블 레이아웃.

```
┌────────────────────────────────────────────────────────────────┐
│ 🎯 HEXSERA ADMIN                              👤 admin [로그아웃] │  ← 상단 헤더
├──────────────┬─────────────────────────────────────────────────┤
│              │  회원 관리                                       │
│  📋 회원관리 │                                                 │
│              │  [ 검색: 닉네임/이메일       ] [🔍]  [+ 추가]   │
│  📊 점수관리 │                                                 │
│              │ ┌────┬───────────┬────────┬────────┬──────────┐ │
│  ⚙️ 설정    │ │ ID │ 이메일     │ 닉네임 │ 역할   │  관리    │ │
│              │ ├────┼───────────┼────────┼────────┼──────────┤ │
│              │ │  1 │ a@a.com   │ admin  │ admin  │ [수정][삭]│ │
│              │ │  2 │ b@b.com   │ 홍길동  │ user   │ [수정][삭]│ │
│              │ │  3 │ c@c.com   │ 닌자   │ user   │ [수정][삭]│ │
│              │ └────┴───────────┴────────┴────────┴──────────┘ │
│              │                                                 │
│              │           [< 이전]  1  2  3  [다음 >]           │  ← 페이지네이션
└──────────────┴─────────────────────────────────────────────────┘
```

**구성 요소**
- 헤더: 어드민 로고 + 로그아웃
- 사이드바: 회원관리, 점수관리, 설정
- 메인: 검색 바 + 추가 버튼 + 테이블 (ID, 이메일, 닉네임, 역할, 수정/삭제)
- 페이지네이션: 하단 페이지 이동

## 페이지 공통 가이드

| 항목 | 값 |
|------|-----|
| 기본 폰트 | Noto Sans KR, sans-serif |
| 기본 배경 | #F9FAFB (연한 회색) |
| 카드 배경 | #FFFFFF |
| 보더 | #E5E7EB |
| 주요 텍스트 | #1F2937 |
| 포인트 컬러 | #4F46E5 (인디고, 핀볼 테마) |
| 위험 버튼 | #EF4444 (빨강, 탈퇴/삭제) |
| 성공 버튼 | #10B981 (초록, 저장) |
| 반응형 브레이크 | xs: 0px / sm: 600px / md: 900px / lg: 1200px |

## 모바일 대응 원칙

- **사이드바**: 모바일(xs/sm)에서 기본 숨김, 햄버거 메뉴로 토글
- **게임 화면**: `transform: scale(0.5)` + `height: 600px` 명시
- **카드 레이아웃**: 모바일에서 단일 컬럼 스택
- **테이블**: 모바일에서 카드형으로 전환 (또는 가로 스크롤)
