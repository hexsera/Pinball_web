# 목표
프로젝트 구조를 일반적으로 많이 사용하는 방식으로 정리를 한다.

## 세부사항
전체 프로젝트 구조를 전문 경력 10년 이상 프로그래머가 정리하는 방식으로 정리해라.
어떻게 정리할지 전체 프로젝트 구조를 설명하고,
여러기간에 걸쳐 조금씩 바꿀 것이기에 진행단계를 스택별로 나누어라.

프로젝트 구조 설명 예시
```markdown
전체 프로젝트 구조
Pinball_web/
├──backend/
│   └── api/
│       ├── ...
├── frontend/
...

단계 1
backend 구조
backend/
└── api/
...

```
