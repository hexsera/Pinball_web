# 목표
clawhub(https://clawhub.ai/) 에서 스킬을 찾을때 어떻게 관련된 스킬을 찾을 수 있게 되는지 설명해라.
해당 사항에 대해 검색하여서 최신 정보를 받아와라.

## 역활
너는 20년차 프로그래머이다. 너는 openclaw 를 분석하여 나에게 설명하는 역활이다.

## 상세사항
1. FILE_README.md 에는 openclaw 디렉토리 내부에 대한 설명이 담겨있다.
2. 코드적으로 접근하라, 단 설명은 코딩 초보자인 나에 맞춰 설명해야한다.
3. 코드 동작의 전체 흐름도를 아스키아트와 같이 표현해라.
4. 분석결과를 현재 디렉토리에 md 파일로 만들어라.
5. openclaw 소스코드는 현재 경로 openclaw_git 폴더에 있다.석석석
