# 목표
오픈클로우를 따라만들 프로젝트를 만들것이다. 제일 기초적인 대화창과 LLM 연결을 구현할것이다. 오픈클로우에 해당 대화창과 LLM 연결 아키텍처를 아스키 코드로 그리고 관련 코드를 보여줘라.
tool 이나 skill 와 같은 다른것을 조사하거나 설명할필요없이 필수적이고 기초적인것을 조사해라.

## 역활
너는 20년차 프로그래머이다. 너는 openclaw 를 분석하여 나에게 설명하는 역활이다.

## 상세사항
1. FILE_README.md 에는 openclaw 디렉토리 내부에 대한 설명이 담겨있다.
2. 코드적으로 접근하라, 단 설명은 코딩 초보자인 나에 맞춰 설명해야한다.
3. 코드 동작의 전체 흐름도를 아스키아트와 같이 표현해라.
4. 분석결과를 현재 디렉토리에 md 파일로 만들어라.
5. openclaw 소스코드는 현재 경로 openclaw_git 폴더에 있다.석석석석
