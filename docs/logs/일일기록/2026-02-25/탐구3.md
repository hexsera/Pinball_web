# 목표
openclaw 에서 openclaw 가 스스로 tool 을 사용하려는 판단을 하고 해당 tool 을 호출하는 작동과정을 아스키 코드로 나타내라.
또한 openclaw 의 tool 호출 결과를 스스로 해석하는 과정과 주고받는 데이터 구조또한 나타내라.

## 주의사항
1. openclaw 디렉토리 (openclaw-src 는 무시) 는 openclaw 의 소스코드가 담겨져있다.
2. opencalw 디렉토리 세부 파일 내용은 FILE_README.md 를 읽으면 파악할 수 있다.
3. 목표를 읽고 결론을 내는 과정에서 직접 코드를 해석하여 결론을 내도록 해라.
4. 분석결과를 현재 디렉토리에 md 파일로 생성해라.
