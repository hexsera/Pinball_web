# 구글 로그인 버튼 디자인 변경 실행계획

## 요구사항 요약

**요구사항**: Login.jsx의 "구글로 로그인" 버튼을 구글 공식 브랜드 가이드라인에 맞는 디자인으로 변경한다.

**목적**: 사용자가 버튼을 보고 즉시 구글 로그인임을 인식할 수 있도록 한다. 기존 테스트(`/구글로 로그인/i` role=button)가 통과되어야 한다.

## 현재상태 분석

- `Login.jsx` 183~191번 줄에 `Button variant="outlined"` 형태로 "구글로 로그인" 버튼 존재
- 현재 버튼: 텍스트만 있고, 구글 로고 없음, 일반 MUI outlined 스타일
- `handleGoogleLogin` 함수: 이미 구현 완료 (`GET /api/v1/auth/google/login` → URL 이동)
- 테스트 파일: `getByRole('button', { name: /구글로 로그인/i })`로 버튼을 찾음 — 버튼 텍스트 유지 필수

## 구현 방법

- 구글 공식 SVG 로고를 인라인 또는 이미지로 버튼 내부 왼쪽에 배치한다.
- 구글 브랜드 가이드라인: 흰색 배경(`#FFFFFF`), 검정 텍스트(`#3c4043`), 테두리 색상(`#dadce0`), 호버 시 연한 회색 배경
- MUI `Button` 컴포넌트의 `startIcon` prop을 사용하거나, `Box`로 직접 레이아웃을 구성한다.
- 구글 SVG 아이콘은 외부 URL 의존 없이 인라인 SVG 컴포넌트로 직접 작성한다.

## 구현 단계


- **무엇을 하는가**: 외부 이미지 파일 없이 구글 공식 4색 로고를 SVG로 직접 렌더링
- 구글 공식 색상 4가지 (파란색, 초록색, 노란색, 빨간색) 각각의 `path`로 구성
- 크기 18×18px로 버튼 텍스트와 비례 맞춤

### 2. 구글 버튼 sx 스타일 변경

```jsx
<Button
  variant="outlined"
  startIcon={<GoogleIcon />}
  sx={{
    width: '300px',
    backgroundColor: '#FFFFFF',
    color: '#3c4043',
    borderColor: '#dadce0',
    textTransform: 'none',
    fontFamily: 'Google Sans, Roboto, sans-serif',
    fontWeight: 500,
    fontSize: '14px',
    '&:hover': {
      backgroundColor: '#f8f9fa',
      borderColor: '#dadce0',
      boxShadow: '0 1px 3px rgba(60,64,67,.3)',
    },
  }}
  onClick={handleGoogleLogin}
>
  구글로 로그인
</Button>
```

- **무엇을 하는가**: MUI Button의 `startIcon`으로 구글 SVG 로고를 왼쪽에 배치, 흰색 배경/검정 텍스트로 구글 공식 스타일 적용
- `textTransform: 'none'`: MUI 기본 대문자 변환 방지 (한글에는 무관하나 명시적으로 원본 텍스트 유지)
- `variant="outlined"` 유지: 테두리 버튼 형태로 구글 가이드라인과 일치
- 버튼 텍스트 "구글로 로그인" 유지: 기존 테스트의 `name: /구글로 로그인/i` 통과 필수
