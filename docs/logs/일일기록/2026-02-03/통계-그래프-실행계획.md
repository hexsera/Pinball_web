# 통계 그래프 실행계획

## 요구사항 요약

**요구사항**: 관리자 통계 페이지에 주별 회원 방문 통계를 보여주는 Line Chart 추가

**목적**: 관리자가 주별 회원 방문 추이를 시각적으로 확인하여 서비스 사용 패턴 파악

## 현재상태 분석

- AdminStatisticsMain.jsx: 통계 페이지 메인 컴포넌트 존재, 현재 "통계 데이터가 여기에 표시됩니다." 메시지만 표시
- package.json: @mui/material, @mui/icons-material 설치됨, @mui/x-charts는 미설치
- MUI X Charts: MUI에서 공식 제공하는 차트 라이브러리, Line Chart 컴포넌트 제공

## 구현 방법

- MUI X Charts 패키지 설치 (`@mui/x-charts`)
- LineChart 컴포넌트 사용하여 회원 방문 통계 시각화
- 임시 데이터로 최근 8주간의 방문자 수 표시 (실제 API 연동은 추후 작업)
- AdminStatisticsMain.jsx 수정하여 Line Chart 추가

## 구현 단계

### 1. @mui/x-charts 패키지 설치

```bash
cd /home/hexsera/Pinball_web/react/main
npm install @mui/x-charts
```
- **무엇을 하는가**: MUI X Charts 라이브러리를 프로젝트에 설치
- npm을 통해 @mui/x-charts 패키지 추가
- Line Chart, Bar Chart 등 다양한 차트 컴포넌트 사용 가능

### 2. AdminStatisticsMain.jsx 수정 (import 추가)

```javascript
import { Box, Typography, Toolbar } from '@mui/material';
import { LineChart } from '@mui/x-charts/LineChart';
```
- **무엇을 하는가**: LineChart 컴포넌트를 import하여 사용 가능하도록 설정
- @mui/x-charts 패키지에서 LineChart 컴포넌트 가져오기

### 3. 임시 데이터 생성

```javascript
function AdminStatisticsMain() {
  const drawerWidth = 260;

  // 임시 데이터: 최근 8주간 방문자 수
  const visitData = {
    xAxis: ['1주차', '2주차', '3주차', '4주차', '5주차', '6주차', '7주차', '8주차'],
    visitors: [320, 450, 380, 520, 600, 470, 560, 610],
  };
```
- **무엇을 하는가**: 차트에 표시할 샘플 데이터 생성
- xAxis: 주차 레이블 (8주)
- visitors: 각 주차별 방문자 수
- 실제 API 연동 전까지 사용할 임시 데이터

### 4. Line Chart 컴포넌트 추가

```javascript
<Box sx={{
  backgroundColor: '#ffffff',
  borderRadius: 1,
  p: 3,
  border: '1px solid #e5e7eb'
}}>
  <Typography variant="h6" sx={{ mb: 2, color: '#1f2937' }}>
    회원 방문 통계 (최근 8주)
  </Typography>
  <LineChart
    xAxis={[{
      data: [1, 2, 3, 4, 5, 6, 7, 8],
      scaleType: 'point',
      label: '주차'
    }]}
    series={[{
      data: visitData.visitors,
      label: '방문자 수',
      color: '#465FFF',
      showMark: true,
    }]}
    width={800}
    height={400}
  />
</Box>
```
- **무엇을 하는가**: Line Chart를 AdminStatisticsMain에 추가하여 주별 방문 통계 시각화
- xAxis: x축 데이터 설정 (1~8, scaleType: 'point'로 점 단위 표시)
- series: y축 데이터 설정 (방문자 수, 라벨, 색상, 마커 표시)
- width/height: 차트 크기 설정 (800x400px)
- 흰색 배경 카드 안에 차트 배치
