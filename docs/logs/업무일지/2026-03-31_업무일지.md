## 주요 변경 사항

### 1. docker-compose.yml 수정

**변경 전:**
```yaml
  fastapi:
      build: ./backend
      container_name: fastapi-server
      restart: always
      working_dir: /code
      volumes:
        - ./backend:/code
      ports:
        - "8000:8000"
      depends_on:
        - postgres
      networks:
        - web
      labels:
        - "traefik.enable=true"
        - "traefik.http.routers.fastapi.rule=Host(`hexsera.com`) && PathPrefix(`/api`)"
        - "traefik.http.routers.fastapi.entrypoints=websecure"
        - "traefik.http.routers.fastapi.tls.certresolver=cloudflare"
        - "traefik.http.services.fastapi.loadbalancer.server.port=8000"

# (하단 volumes 섹션)
volumes:
  html-data-volume:
  mysql-data:
  postgres-data:
```

**변경 후:**
```yaml
  redis:
    image: redis:8.6.2
    container_name: redis-server
    restart: always
    ports:
      - "6379:6379"
    volumes:
      - redis-data:/data
    networks:
      - web

  fastapi:
      build: ./backend
      container_name: fastapi-server
      restart: always
      working_dir: /code
      volumes:
        - ./backend:/code
      ports:
        - "8000:8000"
      depends_on:
        - postgres
        - redis
      networks:
        - web
      labels:
        - "traefik.enable=true"
        - "traefik.http.routers.fastapi.rule=Host(`hexsera.com`) && PathPrefix(`/api`)"
        - "traefik.http.routers.fastapi.entrypoints=websecure"
        - "traefik.http.routers.fastapi.tls.certresolver=cloudflare"
        - "traefik.http.services.fastapi.loadbalancer.server.port=8000"

# (하단 volumes 섹션)
volumes:
  html-data-volume:
  mysql-data:
  postgres-data:
  redis-data:
```

**변경 의도:** 게임 세션 저장용 Redis 서버를 Docker 스택에 추가. fastapi의 depends_on에 redis 추가하여 Redis가 먼저 기동된 후 FastAPI가 시작되도록 보장.

### 3. backend/app/redis_client.py 신규 생성

**추가:**
```python
import redis
import os

redis_client = redis.Redis(
    host=os.getenv("REDIS_HOST", "redis-server"),
    port=int(os.getenv("REDIS_PORT", 6379)),
    decode_responses=True
)
```

**변경 의도:** 앱 전체에서 Redis 연결 객체를 공유하기 위한 클라이언트 모듈. `decode_responses=True`로 응답값을 bytes 대신 str로 자동 변환. 호스트명은 환경변수 우선, 기본값은 docker-compose 컨테이너명(`redis-server`).

## 영향받은 기능

| 기능 | 변경 내용 | 작동 상태 |
|------|----------|-----------|
| Docker 서비스 기동 | Redis 컨테이너 추가 | 정상 |
| FastAPI 서버 기동 | depends_on에 redis 추가, redis 패키지 설치 | 정상 |
| 기존 API 전체 (로그인, 점수, 친구 등) | 변경 없음 — Redis는 신규 추가만 됨 | 정상 (미확인) |
| Redis 연결 (신규) | redis_client.ping() → True 확인 | 정상 |
| Redis 읽기/쓰기 (신규) | set/get/TTL 동작 확인 | 정상 |
