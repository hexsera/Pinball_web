### 2. Pinball.jsx 터치 디바이스 감지 수정

**변경 전:**
```jsx
// 터치 디바이스 감지
useEffect(() => {
  setIsTouchDevice(navigator.maxTouchPoints > 0);
}, []);
```

**변경 후:**
```jsx
// 터치 디바이스 감지 (터치 지원 + 모바일 화면 너비 768px 이하)
useEffect(() => {
  const check = () => setIsTouchDevice(navigator.maxTouchPoints > 0 && window.innerWidth <= 768);
  check();
  window.addEventListener('resize', check);
  return () => window.removeEventListener('resize', check);
}, []);
```

**변경 의도:** `navigator.maxTouchPoints > 0` 단독 조건은 터치스크린 내장 노트북/데스크탑에서도 `true`를 반환해 데스크탑 화면에서 터치 버튼이 표시되는 버그가 있었음. 화면 너비 768px 이하 조건을 추가하여 실제 모바일 화면에서만 버튼이 노출되도록 수정. 화면 크기 변경 시 즉시 재감지하도록 resize 이벤트 리스너도 추가.
