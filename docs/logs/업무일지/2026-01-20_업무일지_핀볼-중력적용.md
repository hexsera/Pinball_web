## 주요 변경 사항

### 1. 중력 설정 (react/main/src/Pinball.jsx:45)
- Engine.create() 옵션의 gravity.y를 0에서 1로 변경
- Matter.js에서 y축 양수 방향이 아래쪽
- gravity.y = 1은 지구 중력과 유사한 물리 효과

**변경 전:**
```javascript
const engine = Engine.create({
    gravity: {
        x: 0,
        y: 0
    }
});
```

**변경 후:**
```javascript
const engine = Engine.create({
    gravity: {
        x: 0,
        y: 1
    }
});
```

### 2. 공 초기 위치 조정 (react/main/src/Pinball.jsx:85)
- 중력 효과를 확인하기 위해 공을 화면 상단 중앙으로 이동
- x: 550 → 350 (중앙 정렬)
- y: 600 → 100 (상단 위치)

**변경 전:**
```javascript
const ball = Bodies.circle(550, 600, 15, { ... });
```

**변경 후:**
```javascript
const ball = Bodies.circle(350, 100, 15, { ... });
```

### 3. 플리퍼 위치 조정 (react/main/src/Pinball.jsx:104, 123)
- 왼쪽 플리퍼 y 좌표: 550 → 1170 (하단으로 이동)
- leftFlipperConstraint pointB: { x: 260, y: 550 } → { x: 260, y: 1170 }
- 공이 떨어져서 플리퍼로 쳐올릴 수 있도록 배치
